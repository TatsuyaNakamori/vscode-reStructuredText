Response Status
2-byte special characters are
misaligned in width (※, etc.)
Use single-byte characters (e.g. *) or
manually adjust the width (add one space).
Scrutinizing letters that are

Cell merging is not supported.
Merge the cells manually after you are done editing
(Note that if you press a button such as Enter in a cell,
all the cells will be split.)
